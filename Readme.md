## Pre-requisites
- Install [Node.js](https://nodejs.org/en/) version 18.0.0

## Getting started
- Clone the repository
```
git clone https://github.com/chinezenwosu/kwado.git
```

- Install dependencies
```
cd <project_name>
npm install
```

- Build and run the project
```
npm start
```

- Navigate to `http://localhost:3000`
